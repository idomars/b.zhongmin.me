HTML5 jQuery图片上传前预览

发表于 2017-01-06 | 更新于 2019-05-06 | 分类于 前端知识

主要是为了实现点击头像是可以上传图片，上传图片后预览头像的功能，Html代码如下：

img标签是用来展示图片的地方，而 file标签是为了上传图片。

//点击头像触发input file click 事件
mui(‘#app’).on(‘click’, ‘#tap’, function(e) {
$(‘#headImgFile’).click();
});

这是mui的绑定事件，给头像绑定点击事件，点击头像触发file标签的点击事件，进行上传图片。

//头像 事件
mui(‘#app’).on(‘change’, ‘#headImgFile’, function(e) {
var objUrl = getObjectURL(this.files[0]) ;
console.log(“objUrl = “+objUrl) ;
if (objUrl) {
$(“#headImg”).attr(“src”, objUrl) ;
}
});

function getObjectURL(file) {
var url = null ;
if (window.createObjectURL!=undefined) { // basic
url = window.createObjectURL(file) ;
} else if (window.URL!=undefined) { // mozilla(firefox)
url = window.URL.createObjectURL(file) ;
} else if (window.webkitURL!=undefined) { // webkit or chrome
url = window.webkitURL.createObjectURL(file) ;
}
return url ;
}

给file标签绑定change事件，通过HTML5 的File API,建立一個可存取到该file的url，放入到img的src进行展示。只能在支持HTML API的浏览器下实现，比如谷歌Chrome和火狐等。
